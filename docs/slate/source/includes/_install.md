# Instalación

Dado que el juego no cuenta con una modalidad ***no-multiplayer***, es necesario contar con el servidor en funcionamiento.<br>

Para poder ejecutar el servidor, se necesita tener instalado la última versión de **[Node JS](https://nodejs.org/es/)**.<br>
Puede descargar la última version aquí **[https://nodejs.org/es/download/](https://nodejs.org/es/download/)**.

Una vez instalado Node JS, debe abrir una términal y ejecutar los siguientes comandos:<br> ***( ver pestaña shell )***.


```shell
# Clonar el repositorio
$ git clone https://github.com/ehopperdietzel/JuegoGrafica.git

# Acceder al directorio raíz
$ cd JuegoGrafica

# Instalar los módulos requeridos
$ npm install

# Ejecutar el servidor
$ node server.js
```
